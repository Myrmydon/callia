📌 VERSION PORTABLE – Aucune installation requise

───────────────────────────────
🇫🇷 UTILISATION
───────────────────────────────
1. Lancez Callia.exe (double-cliquez) depuis le dosier dist
2. Collez ou importez votre texte (Word, .odt, .txt…)
3. Choisissez vos options :
   - Format du livre (A4, A5, A6, KDP…)
   - Police, taille de texte, interligne
   - Alinéas, numérotation des pages, sommaire cliquable
4. Cliquez sur « Générer le PDF »
5. Retrouvez votre livre prêt à imprimer dans le dossier de votre choix

📁 Vous pouvez déplacer ce dossier où vous voulez.
📌 Pour créer un raccourci sur le bureau : clic droit sur `Callia.exe` > Envoyer vers > Bureau.

───────────────────────────────
🇬🇧 HOW TO USE
───────────────────────────────
1. Launch Callia.exe (double-click)
2. Paste or import your text (Word, .odt, .txt…)
3. Choose your layout options:
   - Book size (A4, A5, A6, KDP…)
   - Font, text size, line spacing
   - Indents, page numbers, interactive table of contents
4. Click “Generate PDF”
5. Your print-ready book will be saved in the **output** folder

📁 You can move this folder anywhere you like.
📌 To create a desktop shortcut: right-click `Callia.exe` > Send to > Desktop.
